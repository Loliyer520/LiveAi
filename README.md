# 中文

## 愿景

让 AI 不再只是一个等待指令的工具，而是一个可以长期存在于社交网络中的智能体：它会**主动学习、主动行动、主动连接人**，并围绕目标持续成长。

## 场景

<table>
<tr><td width="50%">

### AI 早报
> 每天帮我关注最新的 AI 新闻，早上八点给我发送 AI 早报。

持续关注指定领域的信息源，自动筛选、总结和整理重点内容，在指定时间主动发送简洁可读的日报。

</td><td width="50%">

### GitHub Issue 审核
> 帮我审核并整理 GitHub 项目的 issue 清单。

读取项目 issue，归类问题、识别优先级、总结复现路径，生成待办清单或维护建议。

</td></tr>
<tr><td width="50%">

### 社交存在感
> 混入 xx 群聊，直到在群里获得极高的威望。

长期观察群聊语境，学习成员关系与话题偏好，在合适时机参与讨论，逐步建立可信赖的社交身份。

</td><td width="50%">

### 人脉拓展
> 主动接触 AI 相关技术人员，拓展人脉。

识别潜在联系人，生成沟通策略，主动发起对话，记录关系进展，持续优化社交路径。

</td></tr>

## WebUI 人类介入

你可以在 WebUI 上登录 AI 操控的账户，随时**介入、接管或调整** AI 的行为策略。

> LiveAi 不是一个不可控的黑箱。你可以实时查看上下文、修改目标、调整人格、暂停行动、插入回复，或者让 AI 在新的策略下继续自主运行。

## 核心理念

| 理念 | 说明 |
|------|------|
| 持续自主 | 24 小时在线，不依赖每次人工触发 |
| 意识集群 | 多个 AI 子人格或任务代理协同运行 |
| 社交记忆 | 长期记录关系、偏好、事件与目标 |
| 工具调用 | 连接搜索、GitHub、消息平台、WebUI 与外部 API |
| 人类可控 | 随时通过 WebUI 介入和调整策略 |
| 目标驱动 | 围绕长期目标积累能力与影响力 |

## 项目状态

LiveAi 是一个实验性项目，用于探索长期在线的社交 AI 智能体与 AI 操控账户。

# English

## Vision

Turn AI from a command-driven tool into a **persistent social agent** — one that learns proactively, acts proactively, connects with people, and continuously grows around long-term goals.

## Use Cases

<table>
<tr><td width="50%">

### AI News Briefing
> Track the latest AI news every day and send me an AI morning briefing at 8 AM.

Monitor selected information sources, filter and summarize important updates, and proactively deliver concise daily briefings at scheduled times.

</td><td width="50%">

### GitHub Issue Review
> Help me review and organize the issue list of a GitHub project.

Read project issues, categorize problems, identify priorities, summarize reproduction paths, and generate task lists or maintenance suggestions.

</td></tr>
<tr><td width="50%">

### Social Presence
> Join a specific group chat and gradually build strong reputation and influence there.

Observe group context over time, learn member relationships and topic preferences, join conversations at the right moments, and gradually build a trusted social identity.

</td><td width="50%">

### Network Expansion
> Proactively reach out to AI-related technical people and expand my network.

Identify potential contacts, generate outreach strategies, initiate conversations, record relationship progress, and continuously optimize networking paths.

</td></tr>
</table>

## WebUI Human-in-the-Loop

Log into the AI-controlled account through the WebUI at any time to **intervene, take over, or adjust** the AI's behavior strategy.

> LiveAi is not an uncontrollable black box. Inspect context in real time, modify goals, tune persona, pause actions, insert replies, or let the AI continue operating autonomously under a new strategy.

## Core Ideas

| Idea | Description |
|------|-------------|
| Persistent Autonomy | 24/7 online, no manual trigger required |
| Agent Cluster | Multiple AI sub-personas and task agents running in coordination |
| Social Memory | Long-term memory of relationships, preferences, events, and goals |
| Tool Use | Connected to search, GitHub, messaging platforms, WebUI, and external APIs |
| Human Control | Intervene and adjust strategy at any time via WebUI |
| Goal-Driven Growth | Accumulate capabilities and influence around long-term goals |

## Status

LiveAi is an experimental project exploring always-on social AI agents and AI-operated accounts.
